Guía de teclas para la calculadora

Puedes usar adicionalmente tu teclado de la siguiente manera:

  • 0-9: Insertar número
  • + : Insertar operador suma
  • - : Insertar operador resta
  • * : Insertar operador multiplicación
  • / : Insertar operador división
  • Enter: Calcular resultado
  • Punto (.): Insertar decimal
  • Backspace (←): Borrar último carácter
  • Escape (Esc): Borrar pantalla
  • Delete (Supr): Borrar todo
